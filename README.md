#Contact Form

<p> The form is a simple contact form saving user details in backend mysql database. On the basis of these details depends the data of first two charts in demo. </p>

#Chart1

<p> Chart#1 is a bar chart displaying count of number of male and female present in database. The data is passed through server side api and ajax is used to show it on client side. </p>

#Chart2

<p> Chart#2 is a pie chart which is showing divsion on basis of messege status. </p>

#Chart3

<p> The bar chart is showing data using client side processing. The bars show the height and width of images from Google image search API. </p>

#Chart4

<p> The chart is same as above. The difference is that API is processed server side to get data. </p>
